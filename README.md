# 🌐 Ganesh Sahu - Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, Bootstrap, and JavaScript** to showcase my skills, projects, and contact information.

---

## 🚀 Live Demo

👉 https://ganeshsahuportfolio.netlify.app/

---

## 📖 Overview

This portfolio represents my profile as a **Full Stack Developer**, highlighting my technical skills, projects, and ability to build responsive and user-friendly web applications.

The focus of this project is on:
- Clean UI/UX
- Responsiveness across devices
- Interactive frontend features using JavaScript

---

## ✨ Features

- ✅ Fully responsive design (Mobile, Tablet, Desktop)
- ✅ Smooth scrolling navigation
- ✅ Typing animation effect (JavaScript)
- ✅ Scroll-to-top button
- ✅ Modern dark theme UI
- ✅ Projects showcase with live links
- ✅ Contact section with external links

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Bootstrap 5  
- **JavaScript:** DOM Manipulation, Event Handling  
- **Tools:** Git, GitHub, Netlify, VS Code
